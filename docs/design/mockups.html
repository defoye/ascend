05

How to export

01
Colours → create an Asset Catalog with one Color Set per token in §2.1/§2.2. Set the Any Appearance value from the Light table and the Dark value from the Dark table. Reference as Color("primary") or wrap in an AscendColor enum.
02
Type → use SwiftUI text styles (§2.3) with Dynamic Type; add .monospacedDigit() on stats & chart numbers.
03
Spacing / radii / elevation → paste §2.4–2.6 into a Spacing/Radius enum and a .cardShadow() ViewModifier.
04
Components → build the §3 list as reusable SwiftUI views (PrimaryButton, VerifiedBadge, StatTile, ProgressChart, …), then compose the screens.
05
The full written spec lives in Ascend-Design-Spec.md — copy that file for the complete tables, principles and interaction notes.
