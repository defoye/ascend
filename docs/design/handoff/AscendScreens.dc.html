Trust is the interface.

Five high-traffic surfaces plus the shared state kit — loading, empty and error — designed once and applied everywhere. Light and dark, real data, one confident teal. Every screen reads like a credible record, not a marketing surface.

01

Coach · Today

The confident home base. Real loading, tappable session + activity rows with drill-in chevrons, and a revenue snapshot that reads as a calm ledger.

02

Coach · Client Detail

The single densest screen. Header + status, stat tiles, program summary, one metric chart, recent entries and notes — with a live message entry point in the nav bar (today a dead “coming soon” alert).

03

Proof Profile

The evidence-first flagship. The teal Verified journey badge (paid mode) always carries a plain-language substantiation line; free mode degrades to a neutral Tracked results badge with no verified claim. We verify that the journey happened — never that the coach caused the result.

04

Consumer · Today

The calm daily surface. One clear thing to do — today's workout — with the next session, a coach nudge, and an honest bodyweight trend. Rest days are a feature, not an empty screen.

05

Consumer · Workout Player

The one delightful screen. Log sets against last time, a live rest timer between them. Success is calm and factual — a logged delta, never confetti. Finishing reads like closing a credible record.

06

Shared state kit

Three states, designed once and applied to every surface above — so loading, empty and error read as one system, not five improvisations. Plus the single success microinteraction the whole product leans on.
